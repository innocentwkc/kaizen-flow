Module: icsParser

This module provides a function to parse ICS (iCalendar) data. It takes raw ICS data, processes it, and converts it into a structured format of calendar events. This is useful for extracting and manipulating calendar data from ICS files programmatically.

Parses ICS (iCalendar) data. This function takes raw ICS data, processes each line, and constructs an array of event objects, each representing a single calendar event. It uses 'moment-timezone' for date-time manipulations.
